# Invoice Generator

## Description
A simple and efficient invoice generator that allows users to create, manage, and export professional invoices. Designed for businesses and individuals who need a fast and reliable way to handle billing.

## Features
- Create and manage invoices
- Add customer details
- Dynamic item listing (price, quantity, total)
- Automatic calculations (subtotal, tax, total)
- Generate downloadable/printable invoices (PDF)
- Clean and user-friendly interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: .NET Core (if applicable)
- Database: SQL Server (if applicable)
